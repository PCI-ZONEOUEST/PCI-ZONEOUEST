Informations sur les commandes Tera Term

Pour Cisco :

Voici quelques commandes utiles pour Tera Term avec des appareils Cisco :

  • enable : Passer en mode privilégié.
  • configure terminal : Accéder au mode de configuration globale.
  • show ip interface brief : Afficher un résumé des interfaces IP.

Pour Juniper :

Voici quelques commandes utiles pour Tera Term avec des appareils Juniper :

  • cli : Passer en mode CLI.
  • edit : Accéder au mode de configuration.
  • show interfaces terse : Afficher un résumé des interfaces.
